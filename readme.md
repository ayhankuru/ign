Command line created ignore file!


## Install

```
npm install -g ign
```

## Create ignore file

```
ign create

```
## İgnore file add

```
ign add

```


## İgnore cache file create

```
ign cache

```

## İgnore cache apply 

```
ign apply

```

## İgnore delete file 

```
ign delete

```
